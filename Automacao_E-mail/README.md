# Automação de Sistemas e Processos com Python

### Desafio:
(cenário hipotético)

Todos os dias, o nosso sistema atualiza as vendas do dia anterior.
O seu trabalho diário, como analista, é enviar um e-mail para a diretoria, assim que começar a trabalhar, com o faturamento e a quantidade de produtos vendidos no dia anterior.

Para isso, o sistema utilizado com a fonte de dados é o Google Drive, a ideia então é acessar o sistema, baixar o arquivo, analisar os indicadores e enviar essa análise por e-mail.


Para resolver isso, vou usar o pyautogui, uma biblioteca de automação de comandos do mouse e do teclado, o pandas para analisar os dados e algumas bibliotecas auxiliares.
